# photos/emojis/ — generated cutouts (cars + danger stickers)

> Rule: update this file in the same commit as any feature change here.

True-alpha PNG cutouts generated by `python src/make_faces.py` — **do not
edit by hand**, regenerate instead. Embedded as data URIs in
`src/photos.json` (`cars` and `terros` keys).
